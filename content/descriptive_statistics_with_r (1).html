Descriptive Statistics with R¶

Bernard Ade

The goal of Descriptive Statistics is to summarize and describe a dataset. In this post, I will demonstrate how to use R to perform Descriptive Statistics on a small Dataset. My aim is to keep things simple and focus on illustrating R functions and packages such as dplyr and ggplot.

Lets explore the mpg dataset to gain some insight.

Dataset¶

The data that we will use for this excercise is called mpg which comes installed with R. We can learn some basic characteristics by running ?mpg

In [20]:
?mpg
mpg {ggplot2}R Documentation

Fuel economy data from 1999 and 2008 for 38 popular models of car

Description

This dataset contains a subset of the fuel economy data that the EPA makes available on http://fueleconomy.gov. It contains only models which had a new release every year between 1999 and 2008 - this was used as a proxy for the popularity of the car.

Usage

mpg

Format

A data frame with 234 rows and 11 variables

  • manufacturer.

  • model.

  • displ. engine displacement, in litres

  • year.

  • cyl. number of cylinders

  • trans. type of transmission

  • drv. f = front-wheel drive, r = rear wheel drive, 4 = 4wd

  • cty. city miles per gallon

  • hwy. highway miles per gallon

  • fl.

  • class.

[Package ggplot2 version 2.1.0 ]

Central Tendency¶

During the process of understanding data, one of the important goals is to obtain a figure that describes the center of a set of values. Lets describe some common descriptive statistical approaches for summarizing the central tendency using mode, median, mean, range and quartiles.

Histogram¶

Before we dive into obtaining the summaries for various variables within our dataset, lets visualize the distribution of values within each variable to understand the data visually.

In [4]:
ggplot(mpg, aes(hwy)) +
    geom_histogram(col="red", fill="red", alpha=0.3, bins = 20) +
    xlab("Highway Miles per Gallon") +
    theme_bw()
In [5]:
ggplot(mpg, aes(cty)) +
    geom_histogram(col="blue", fill="blue", alpha=0.3, bins = 20) +
    xlab("City Miles per Gallon") +
    theme_bw()
In [6]:
ggplot(mpg, aes(displ)) +
    geom_histogram(col="gray", fill="gray", alpha=0.3, bins = 20) +
    xlab("Engine Displacement in Litres") +
    theme_bw()

Mean¶

The mean also commonly referred to as average summarizes the central tenedency of a set.

In [9]:
## we can obtain the mean by running the mean() function
mean(mpg$hwy)
23.4401709401709

Median¶

The median is the middle value of a set after it has been sorted from high to low.

In [10]:
#we can obtain the median by running the median() function
median(mpg$cty)
17

Range¶

The range retuns the minimum and maximum value in a set. An alternative way to obtains these values is using the max and min function.

In [11]:
# we can obtain the range by running the range() function
range(mpg$hwy)
  1. 12
  2. 44
In [12]:
min(mpg$hwy)
12
In [13]:
max(mpg$hwy)
44

Variance¶

The variance describes the spread of the data and measures how much a variables differs from the mean.

In [16]:
var(mpg$hwy)
35.4577785114266

Correlation¶

Correlation is useful for evaluating the relationship between two or more variables. Suppose we wanted to evaluate the relationship btw engine displacement and city miles per gallon. The cor function in R returns a value that describes the relationship between the engine displacement and the city miles per gallon. The negative correlation suggests that the bigger the engine diplacement the less fuel efficient the vehicle.

In [17]:
cor(mpg$displ, mpg$cty)
-0.798523968934855
In [18]:
ggplot(mpg, aes(x = displ, y = cty)) + geom_jitter() + geom_smooth(method=lm)
